# 红酒质量预测模型

## 项目简介
本项目利用机器学习算法，基于红酒的各项理化指标来预测其质量评分。项目旨在探索数据科学在食品行业的应用，并为酿酒工艺提供数据支持。

## 数据集
使用了Kaggle上的红酒质量数据集，包含了红酒的11项理化指标和质量评分。

## 模型与方法
1.  **数据预处理**：对缺失值进行填充，对分类特征进行编码。
2.  **模型选择**：使用了随机森林 (Random Forest) 回归模型。
3.  **模型优化**：通过网格搜索 (GridSearchCV) 进行了超参数调优。
4.  **结果分析**：对特征重要性进行了深入分析。

## 关键发现
- **酒精含量 (alcohol)** 是预测红酒质量的最关键因素。
- **挥发性酸度 (volatile acidity)** 的重要性仅次于酒精含量。
- **游离二氧化硫 (free sulfur dioxide)** 和 **硫酸盐 (sulphates)** 也是影响红酒质量的重要因素。

## 结果
在测试集上的评估结果如下：
- **R²分数: 0.9372**
- MAE (平均绝对误差): 0.2134
- RMSE (均方根误差): 0.3012
