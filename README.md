# 红酒质量预测模型

## 项目简介
本项目利用机器学习算法，基于红酒的各项理化指标来预测其质量评分。

## 数据集
使用了 Kaggle 上的红酒质量数据集 (winequalityN.csv)，包含了红酒的 11 项理化指标和质量评分。

## 模型与方法
1.  **数据预处理**：对缺失值进行填充，对分类特征进行编码。
2.  **模型选择**：使用了随机森林 (Random Forest) 回归模型。
3.  **模型优化**：通过网格搜索 (GridSearchCV) 进行了超参数调优。

## 结果
在测试集上的评估结果如下：
- R²分数: 0.5573
- MAE (平均绝对误差): 0.4060
- RMSE (均方根误差): 0.5625
